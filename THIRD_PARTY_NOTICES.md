# Third-Party Notices

PDF Vault orchestrates the following open-source tools. When distributed via Docker, some are bundled in the container image.

| Tool | Purpose | License |
|---|---|---|
| [qpdf](https://github.com/qpdf/qpdf) | Merge, split, encrypt, repair | Apache 2.0 |
| [Ghostscript](https://www.ghostscript.com/) | PDF compression | AGPL-3.0 |
| [Poppler](https://poppler.freedesktop.org/) | PDF to image, extract images | GPL-2.0 |
| [OCRmyPDF](https://github.com/ocrmypdf/OCRmyPDF) | OCR, PDF/A | MPL-2.0 |
| [Tesseract](https://github.com/tesseract-ocr/tesseract) | OCR engine | Apache 2.0 |
| [Gotenberg](https://gotenberg.dev/) | Office/HTML to PDF | MIT |
| [img2pdf](https://gitlab.mister-muffin.de/josch/img2pdf) | Images to PDF | LGPL-3.0 |
| [pypdf](https://github.com/py-pdf/pypdf) | PDF manipulation (Python) | BSD-3-Clause |
| [ReportLab](https://www.reportlab.com/) | Watermarks, page numbers | BSD |
| [pdf2docx](https://github.com/dothinking/pdf2docx) | PDF to Word | AGPL-3.0 |
| [FastAPI](https://fastapi.tiangolo.com/) | API framework | MIT |
| [Next.js](https://nextjs.org/) | Web UI | MIT |

When using GPL/AGPL components via CLI in a Docker container, comply with their respective license terms when redistributing images.
